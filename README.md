### Introduce

博客项目，使用hexo，页面部署到leeFengHuo.github.io。

### Install

```
npm install hexo-cli -g
git clone https://github.com/leeFengHuo/myBlog.git
cd myBlog/blog/
npm install
```

### Usage

文章写在`myBlog/blog/source/_posts/`中，其他操作（`myBlog/blog/`目录下执行命令）：
* 本地预览：hexo s
* 生成文件：hexo g
* 部署到leeFengHuo.github.io：hexo d
